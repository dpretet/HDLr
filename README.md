# Install

- Grab a ready-to-use binary in the repo

- Install with your package manager like:

```bash
brew install hdlr
```

- Build with the Python flow

```bash
python3 -venv venv
source venv/bin/activate
pip install -f requirements.txt
pip install -e .
```
